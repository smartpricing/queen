# Webapp Features

Complete feature tour of the Queen MQ dashboard.

## Dashboard

Real-time metrics:
- Messages per second
- Requests per second
- Active queues
- Consumer groups
- System health

## Queues

- List all queues
- Create new queues
- Configure queue settings
- View queue statistics
- Monitor queue depth
- Delete queues

## Messages

- Browse messages
- Search by transaction ID
- Filter by queue/partition
- View message payload
- See message status
- Track message lifecycle

## Consumer Groups

- List all groups
- View group positions
- Monitor lag
- Reset group positions
- Track consumer activity

## Traces

- Visual trace timelines
- Message flow across queues
- Performance analysis
- Bottleneck identification
- Debug workflows

## Analytics

- Historical charts
- Throughput trends
- Queue comparisons
- Performance metrics
- System insights

## System

- Server health
- Database status
- Connection pools
- Thread usage
- Memory metrics
