# Queen MQ Documentation Website

This is the official documentation website for Queen MQ, built with [VitePress](https://vitepress.dev/).

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Structure

- `/` - Home page with features and quick start
- `/guide/` - Getting started guides and core concepts
- `/clients/` - Client library documentation
- `/server/` - Server installation and configuration
- `/webapp/` - Web dashboard documentation
- `/proxy/` - Proxy server documentation
- `/api/` - HTTP and WebSocket API reference

## Theme

The site uses a custom dark theme matching the Queen webapp color scheme:
- Primary: Blue (#3b82f6)
- Secondary: Indigo (#6366f1)
- Accent: Purple (#a855f7)
